# Запуск:
- включаем виртуальное окружение 
- ставим зависимости
- запускаем программу 
    uvicorn main:app --reload --port 8000
  

# описание связей

лид - ему по идее можно назначит несколько контактов(опционально)
в контактах одному оператору можно назначить много лидов(не больше лимита)
в контактах указано из каких источников пришли лиды (в первый раз)
у одного оператора может быть несколько весов (по одному на источник).

## Добавление оператора

     -H "Content-Type: application/json" \
      -d '{"name": "Анна", "active_status": true, "workload_limit": 10}'
